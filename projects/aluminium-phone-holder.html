Aluminium Phone Holder

Machining Project | Aluminium Fabrication

Overview

This project involved manufacturing a functional phone holder from aluminium using workshop machining processes. The objective was to produce a stable stand capable of supporting a mobile phone at a comfortable viewing angle.

The component was produced using a combination of milling, drilling and finishing processes. Dimensional accuracy and surface quality were checked using precision measuring tools throughout the process.

The project developed skills in aluminium machining, safe machine operation and workshop finishing techniques.

Project Specifications

Material Aluminium
Processes Drilling, Bending
Tools Used Pillar Drill, Rivet Gun, Box and Pan Folder
Purpose Mobile Phone Stand
